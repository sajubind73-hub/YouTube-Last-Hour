# YouTube Last Hour Videos

This repository contains a simple HTML page that displays YouTube videos from the last hour. You can open the `Last-hour.html` file in your browser or host it using **GitHub Pages**.

## How to Use

1. Open the `Last-hour.html` file in your web browser.
2. If using GitHub Pages:
   - Go to **Settings â†’ Pages** in this repository.
   - Select **Branch: main** and **Folder: / (root)**.
   - Save and wait a few minutes for your live link.
3. Access the page via the GitHub Pages URL.

## Features

- Displays YouTube videos uploaded in the last hour.
- Lightweight and easy to use.
- Mobile and desktop friendly.
